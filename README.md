![Python](https://img.shields.io/badge/Python-3.x-blue)
![Flask](https://img.shields.io/badge/Flask-2.x-green)

> OSINT automation tool built with Python & Flask

## 🔍 Features

- **Domain Recon** — WHOIS, DNS records, subdomains via crt.sh, HTTP headers
- **IP Investigation** — Geolocation, reverse DNS, network info
- **Username Lookup** — Check 20+ platforms simultaneously
- **Email Analysis** — Format validation, MX records, provider detection

## 🚀 Installation
```bash
git clone https://github.com/Mahmd-75/Phantom.git
cd Phantom
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

## 🛠️ Stack

- Python 3
- Flask
- python-whois
- dnspython
- requests / BeautifulSoup4
